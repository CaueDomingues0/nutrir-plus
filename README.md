# Nutrir+ : Sistema Integrado de Gestão de Doações e Beneficiários

## Descrição do Projeto

O Nutrir+ é um projeto extensionista desenvolvido para a Associação Mãos que Alimentam (AMA). O sistema visa centralizar a gestão de estoque e beneficiários, substituindo processos manuais por uma plataforma web, reduzindo desperdícios e otimizando a logística.

## Estrutura do Repositório

- **/docs**: Documentação técnica (Relatório Final PIE I).
- **/backend**: API REST (Node.js + Express).
- **/frontend**: Interface do usuário (React + Tailwind CSS).
- **/database**: Scripts de modelagem (PostgreSQL).

## Instruções de Instalação

1. Clone este repositório.
2. Configure o arquivo .env com as variáveis de banco de dados.
3. Instale as dependências com 
pm install.
4. Inicie as aplicações com 
pm run dev (backend) e 
pm start (frontend).

## Tecnologias Utilizadas

- **Backend**: Node.js, Express
- **Frontend**: React, Tailwind CSS
- **Database**: PostgreSQL
